(一)自学从0到1，纯前端全栈免费个人博客系列教程之 hexo + github + leancloud + fluid

本文最后更新于：2022年7月12日 下午


小波原计划用 vuePress + github 来搭建一个免费博客，但是后来发现 hexo + github + leancloud 也挺香，遂下定决心完成 廿壴(GANXB2)[小波分享圈子] 的第三次改版，也终于实现了纯前端全栈，和 wordpress 说再见了…

操作系统：win11 21H2 22000.675

git：git version 2.36.1.windows.1
hexo：^5.0.0
hexo-theme-fluid：^1.9.0

🤔 第一步：安装 node 和 git

  • 有梯子的童鞋就直接去上面小波已经准备好的各自官网地址下载 ☝☝
  • 无梯子的童鞋就去上面的官网链接最后一个 “国内镜像官网”下载即可 （全程一路 下一步 即可，如需更详尽的软件安装中英对照图则给小波留言…）

安装成功后在终端中查看，如图：

🤔 第二步：安装 hexo

终端最终提示 Hexo is running at http://localhost:4000 . Press Ctrl+C to stop. 表示成功！复制地址到浏览器地址后打开预览即可。

🤔 第三步：github 注册帐号后并建一个空仓库

3.1. 注册好 github 帐号，如图点击第二个 your repositories

3.2. 然后再点击按钮 New

3.3. 这里需要填入的仓库名字需要和你的帐号用户名一致，格式必须如此： yourname.github.io

3.4. 然后点击右上角头像，点击下拉菜单中的 settings，刷新页面后，再点击左边菜单最后的 Developer settings，点击 按钮 Generate new token

3.5. 这里记得要先把 token 复制到其他地方保存好，页面刷新则再看也不见了。（为博客发布插件使用）

注意勾选上： repo 和 delete_repo ，时间根据自己情况改长一点也可以

🤔 第四步：leancloud 建号，并创建应用

　登录小波上面提供的leancloud官网，注册帐号 ，创建应用。（备用后面开启主题的统计，评论，未来扩展）

  • 国际版 如果没有购买域名并国内备案，或者买的不需要备案的域名，则选择国际版

  • 国内版 如果是国内备案的域名则进入应用后点击左侧菜单–> 设置–>域名绑定


注意：
参考 leancloud文档[10] 去域名管理，把 leancloud 上面的 DNS 解析别名一下即可

🤔 第五步：安装 hexo 主题 fluid、发布推送
